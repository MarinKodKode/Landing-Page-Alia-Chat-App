¿Qué es Alia?

Alia es una aplicación móvil de mensajería instantanea desarrollada para los dispositivos con sistema operativo android. La versión de compilación es Android SDK-30 y la versión mínima de compilación es Android SDK-17. Puede ser ejecutada en el 97% de todos los dispositivos android en circulación actualmente.

Tecnologías usadas

  • Lenguaje de programación

    Alia esta desarrollada en lenguaje Java, usa el patrón de diseño MVVM en su contrucción. Para las interfaces o 'layouts' usa lenguaje XML así como librerías de diseño y la integración de apis de Google.

  • Servicios de backend

    Firebase es el servicio de backend de Alia, integra el uso de APIs, como Google Auth para servicios de autenticación, Google Cloud Messaging para los servicios de mensajería instantanea y para las notifiaciones en tiempo de reposo de la aplicación.
    Usa una base de datos NoSQL basada en documentos JSON en arquitectura de arbol para la modularización de datos.

  • Librerías de terceros

    Principalmente para el diseño UI usa Material Design y integración de la libreria Lottie. Librerías que no existen de manera nátiva en Android.

Acerca del código

El proyecto Alia en su versión v01 tiene 10 recursos xml tipo layout, 16 clases Java que sirven como clases pricipales, adapters, modelos, etc y 3 clases Java que funcionan como interfaces.

  • 1.0 Versión
  • 16 Clases Java
  • 10 Ficheros XML

  • Integración de Firebase

  • API Google Cloud Messaging

Alia esta construida siguiendo el patrón de diseño MVVM, durante el desarrollo optamos por el uso de DataBinding puesto que permite trabajar de manera mucho más fluida haciendo las conexiones entre archivos XML y las clases de Java. Por otro lado contiene un modelo que captura las variables de los documentos JSON de Firebase, esto hace posible añadir más funcionalidades sin necesidad de modificar el proyecto entero. Toda la aplicación está modularizada por lo que el mantenimiento del código es simple y eficiente. Debido al manejo de los estados por los que la aplicación es ejecutada (onCreate, onResume, etc.) es posible añadir funcionalidades como el "En línea" que notifica a los usuarios cuando se encuentran haciendo uso de la aplicación.
